

Plot widget features

* multiple strips (shared x)
* Axes with major/minor tick spacing/range
* major/minor grid (on/off)
* tick label string formatting
* linear/log axis
* Y fit, X scroll/zoom
* legend/signal list
* Markers
  * horizontal line
  * vertical line
  * marker at signal point
  * delta marker (X/Y)
* Cairo output to PDF/SVG/PNG/...
